Clasa SQL_DB

Clasa de lucru cu baza de date.

Subclasa: AG_Operation
Recomand folosirea functiilor clasei in locul simplelor functie de gestionare a bazei de date. Acest lucru face mult mai usoara schimbarea tipului bazei de date sau a unor functii care s-au depreciat intre timp.



Functia sql_connect()

....

// stabilim conexiunea la baza de date
SQL_DB::sql_connect();

Functia sql_querry($query)

Return boolean

Realizeaza o cerere catre baza de date. Alternativa pentru mysql_query().
Returneaza TRUE daca s-a realizat cererea iar FALSE in caz contrar.

SQL_DB::sql_querry("UPDATE `table` SET `nume` = 'Adrian' WHERE `id` = '1' LIMIT 1");

Functia sql_close()

Return boolean

Inchide conexiunea la baza de date. Alternativa pentru mysql_close().
Returneaza TRUE daca s-a realizat cu succes iar FALSE in caz contrar.

// inchidem conexiunea la baza de date
SQL_DB::sql_close();
